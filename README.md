# Chip interpretor project

This is an interpretor for some chip8 roms in C.

## Getting started

### Compilator

* gcc and make

### Dependencies

* SDL2

### How to start

In cmd, type :

````
make
````

Then, in the folder of the interpretor, you will have an file named "chip8.exe"
If you have this file go in cmd and type :

````
chip8 <rom_path>
````
